# Structure Docs

Structure Docs 是 structure 开源社区的官方文档站点，基于 VitePress 构建，旨在为开发者和企业提供全面的生态规范、开源产品说明、集成与运维方案、行业技术栈和 AI 相关内容。

## 项目定位

Structure 是一个面向开发者的开源社区，也是一套企业级开发生态规范，聚合了丰富的开源框架和行业解决方案，助力高效开发与协作。

## 核心产品

- **Structure Boot**：企业级开发脚手架，标准化项目结构与最佳实践
- **Structure Cloud**：云原生应用开发与运维平台，支持微服务与 DevOps
- **生态集成**：与 Ruoyi、宇道源码等主流开源项目深度集成

## 社区目标

- 建立开放、协作、创新的开发者社区
- 推动结构化、标准化的企业级开发生态
- 提供丰富的开源产品与行业解决方案
- 促进技术交流与知识共享

## 快速开始

1. 克隆本仓库
2. 安装依赖并启动 VitePress 本地服务
3. 按照文档结构完善内容

```bash
git clone <repo-url>
cd structure-docs
pnpm install
pnpm docs:dev
```
